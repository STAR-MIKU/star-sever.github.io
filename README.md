# SHOOTING STAR Edge 插件

一个美观的新标签页搜索应用，支持个性化壁纸和多种搜索功能。

## 功能特性

- 🎨 美观的搜索界面
- 🔍 实时搜索建议
- 📚 搜索历史记录
- 🖼️ 个性化壁纸设置
- 📱 响应式设计，适配各种屏幕尺寸

## 安装步骤

1. 打开 Edge 浏览器
2. 访问 `edge://extensions/`
3. 开启右上角的「开发人员模式」
4. 点击「加载已解压的扩展」
5. 选择本项目文件夹

## 使用说明

- **搜索功能**：在中心搜索框输入关键词，按回车键进行搜索
- **搜索建议**：输入时会显示实时搜索建议
- **搜索历史**：自动保存最近10条搜索记录
- **壁纸设置**：通过右下角的壁纸预览区域设置个性化壁纸
- **设置面板**：通过左侧边栏访问各种设置选项

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- Chrome Extension Manifest V3

## 注意事项

- 首次安装需要开启开发人员模式
- 插件将替换您的默认新标签页
- 搜索历史和壁纸设置会保存在本地存储中
